## English

### Version 1.3.0

#### ğŸ‰ New Features

- **Dynamic Tray Window Height Adjustment**: The tray window now automatically adjusts its height based on the number of connected devices (1 device: 230px, 2 devices: 350px, 3+ devices: 460px), providing an optimized user experience
- **Device Information Display Optimization**:
  - Merged "Device" and "Mount Point" into "Device Mount Point" display
  - Adjusted display order: Hard drive capacity is displayed first, followed by device mount point
  - Optimized capacity display format to "Used/Total" (no spaces)
- **Internationalization Enhancement**: Device information in the tray window ("Available", "Total", "Usage Rate") is now fully internationalized, supporting multi-language switching

#### ğŸ”§ Improvements

- **Unified Window Configuration Management**: All window size configurations have been unified to `src/config/window-config.ts` for easier maintenance and management
- **Code Structure Optimization**: Device information rendering logic has been consolidated into shared utility functions to avoid code duplication
- **UI Text Updates**: Changed "Auto Read-Write Access" to "Auto Read-Write" throughout the application

#### ğŸ› Bug Fixes

- **Packaging Issue Fix**: Fixed the error where the `window-config` module could not be found after packaging
- **Dependency Detection Fix**: Fixed the issue where Homebrew and MacFUSE could not be detected in the packaged version. The application now correctly merges the PATH environment variable
- **Debug Information Cleanup**: Removed all debugging `console.log`, `console.warn`, and `console.debug` statements, retaining `console.error` for error tracking

#### ğŸ“¦ Technical Improvements

- Optimized `process.env.PATH` handling to ensure system commands can be correctly found in packaged environments
- Improved IPC communication with new `get-window-size-config` and `adjust-tray-window-height-by-device-count` handlers
- Unified window configuration import and export methods
